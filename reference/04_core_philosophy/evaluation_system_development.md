# Evaluation System Development

## Overview

## Why a Personal Evaluation System?

### The Problem with External Systems
> "The school's evaluation system is 'order and output,' while your logic is 'perception and interest.' In exam-oriented education, students are 'fuel,' not 'observers.'"

External systems:
- Measure compliance, not creativity
- Value obedience over curiosity  
- Create anxiety through comparison
- Don't account for personal growth trajectories

### The Need for Internal Standards
> "You feel you need a 'unique evaluation system to protect your journey' - this is absolutely right. If you use the school's 'score/obedience' system to evaluate yourself, you'll always be a failure."

## Core Principles (Under Development)

### 1. "Less is More" (少即是多)
Following Taleb's philosophy, the system should be minimal but comprehensive:
- Reduce all evaluation to 3 fundamental "axioms" (like RGB primary colors)
- One sentence should be able to judge a day's quality
- Avoid complex checklists

### 2. Via Negativa (Negative Approach)
> "Value = Behavior - Mediocre Compliance"

- Define value by what you reject, not just what you accept
- If an action only serves school evaluation, its value approaches zero
- If an action protects "interest," it's high value

### 3. The Barbell Strategy
Balance two extremes:
- **Extreme Stability**: Anki, anatomy basics - non-negotiable foundations
- **Extreme Randomness**: Creative exploration, philosophical inquiry
- **Avoid the Middle**: Mediocre,不痛不痒 activities

## Potential Axioms (Fermenting)

### Candidate 1: "New Irreversible Cognition"
> "If your evaluation system had only one red line, what would it be called?"

**Answer**: "Did today produce any new, irreversible cognition?"

If not, the day was just "noise" (杂讯) in Taleb's terms.

### Candidate 2: "Interest Protection/Creation"
Does this action:
- Create new "interest"?
- Protect existing "interest" from erosion?
- Reduce vulgarity/mediocrity?

### Candidate 3: "Information Entropy"
> "An indicator called 'Information Entropy' - how much unpredictable new stuff did I intake/create today?"

Measure days by novelty and surprise, not routine completion.

## The "Three Primary Colors" Model

QolarBear wants his evaluation system to work like RGB:
- **3 core axioms** that can generate all conclusions
- Mix them to "color" any activity or day
- **High purity colors** = high value days
- **Gray/brown mud** = low value days

### Possible Color Mixing
- Red (Creation) + Blue (Learning) = Purple (Growth)
- Yellow (Rest) + Red (Creation) = Orange (Recharge for creation)
- All three = White (Optimal day)
- None = Black (Wasted day)
